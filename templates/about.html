About Our Project

This project aims to detect profane words in text using deep learning. It is built with Flask for the backend and Firebase for storing the details of users to give their feedback. We integrated TinyBERT model to classify text as "Profane word detected" or "Profane word not detected". The interface is designed using Bootstrap, ensuring a responsive and user-friendly experience.

About TinyBERT

TinyBERT is a lightweight, distilled version of BERT designed for NLP tasks, offering efficient text understanding with reduced computational cost while maintaining strong performance in various classification problems, including profanity detection. The advantages of TinyBERT are mentioned below

Less Overfitting – Unlike Decision Trees and Logistic Regression, which may overfit on complex textual data, TinyBERT generalizes better by learning deep contextual representations.

Contextual Understanding - Unlike TF-IDF and Multinomial Naïve Bayes, which rely on word frequency, TinyBERT captures word meanings and relationships in context.

Higher Accuracy - Traditional models like Logistic Regression and Decision Trees struggle with complex language, while TinyBERT excels in nuanced text classification.

Efficient for Large Datasets – Unlike TF-IDF-based approaches, which scale poorly with vocabulary size, TinyBERT efficiently processes large datasets with fewer computational resources.

Our goal is to help create a safer online environment by providing an easy-to-use tool for analyzing text content.
